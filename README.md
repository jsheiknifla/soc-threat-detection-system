# SOC Threat Detection System

## 📌 Overview
A real-time Security Operations Center (SOC) system that:
- Collects logs
- Detects attacks (brute force, suspicious login)
- Sends alerts

## 🛠️ Tech Stack
- ELK Stack
- Python

## 🚨 Features
- Real-time monitoring
- Attack detection
- Alert system

## ▶️ How to Run
1. Install requirements
2. Run main.py
3. # soc-threat-detection-system
Real-time SOC threat detection using ELK + Python
